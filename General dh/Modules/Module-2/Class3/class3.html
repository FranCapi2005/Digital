Operador de Asignación (=), sirve para dar valor a una variable y que pueda almacenar su información. Operadores Aritméticos: El símbolo (+) nos permite realizar sumas El símbolo (-) nos permite realizar restas El símbolo (*) nos permite realizar multiplicaciones El símbolo (/) nos permite realizar divisiones Js nos da un Operador Aritmético especial llamado Modulo (%) y sirve para retornar el resto o residuo de una division. Ejemplo: 5%2 = 1 4%2 = 0 Operadores de Incremento y Decremento Objetivo: restar o sumar 1 unidad a un nº determinado. -Incremento: ++ -Decremento: - - Operadores de Comparación -Simple: == (preguntara si un valor es igual a otro) -Estricta: (también preguntara si un valor es igual a otro y ademas pregunta si el tipo de dato es el mismo) Ejemplo: 5 === “5” // False (uno es un number y otro es string) 5>3 // true 15>=26 // false 4<20 // true 4<=2 // false (Antes va el símbolo < o > y después =) Operador Concatenación Se utiliza con el símbolo (+), y sirve para unir 2 o mas cadenas de texto en 1 sola.
