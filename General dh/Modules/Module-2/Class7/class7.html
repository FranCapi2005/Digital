Curso N.7 Curso Programación - Web Full Stack Clase 26-09-2023 *Ciclos FOR Partes escenciales para ejecutar el ciclo "for": - variable incializadora - condicion a evaluar - modificador de la variable inicializadora sintaxis: for () {} - variable incializadora: esta parte del ciclo define en que numero vamos a empezar a iterar - condicion a evaluar: define cuantas iteraciones hara ese mismo ciclo y si se cumple ejecutara las sentencias presentes dentro de las mismas llaves - modificador de la variable inicializadora: permite incrementar (++) o decrementar (--) dicha variable segun la necesidad La condicion a evaluar en algun momento tiene que dar una respuesta negativa *Ciclos WHILE Posee una estructura similar al los condicionales if/else (palabra reservada + Condicion entre parentesis()). Si la condicion es verdadera, este bucle/ciclo, continuara con nuestras indicaciones y si es falsa este se detiene Sin embrago el WHILE LOOP revalúa esa condicion repetidas veces y ejecuta su bloque de codigo hasta que la condicion deje de ser verdadera
