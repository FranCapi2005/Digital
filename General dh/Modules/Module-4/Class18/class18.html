Curso N.18 Curso Programación - Web Full Stack Clase 07-11-2023 *CSS Css es un complemento el cual sirve para dar estilos al front-end de una pagina web, se puede cambiar fondos, tipografia, interlineados, generar animaciones y transiciones. Estilos Basicos: font-family font-size font-style font-weight text-align text-decoration line-height color opacity background-color background-repeat background-image background-position background-attachment background-size *Colores Los siguientes formatos se pueden aplicar a cualquier propiedad de CSS que reciba color: -NOMBRE ej: "purple", "orange", etc... -HEXADECIMAL ej: "#FFF" o "#FFFFFF" -RGB (R.G.B == RED.GREEN.BLUE) ej: rgb(255, 100, 50) -RGBA (R.G.B.A == RED.GREEN.BLUE.OPACITY) ej: rgba(255, 50, 125, 0.5) *BACKGROUND-COLOR p { background-color: tomato } p { background-color: #3459ff } p { background-color: rgb(12, 34, 32) } *BACKGROUND-IMAGE body { background-image: url('../img/bici.jpg') } *BACKGROUND-REPEAT body { background-repeat: repeat-x }
