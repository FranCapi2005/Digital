Curso N.2 Curso Programación - Web Full Stack Clase 07-09-2023 PARTE ASINCRONICA *Comandos Básicos de la Terminal: (https://www.hostinger.es/tutoriales/linux-comandos) >_ ls [List] Muéstralos archivos de la carpeta en la que estamos >_ pwd [PrintWordDirectory] Muestra la ruta en la que estamos >_ cd NombreDeLaCarpeta [ChangeDirectory] Nos permite retroceder una carpeta >_ mkdir NombreDeLaCarpeta [MakeDirectory] Crea una carpeta con el nombre que le damos >_ touch nombreArchivo.extesion Nos permite crear un archivo. Ej: prueba.txt >_ rm nombreArchivo.extension [Remove] Elimina el archivo que le indiquemos >_ mv nombreAnterior nombreNuevo [Move] Cambia el nombre de un archivo por el nuevo que le indiquemos >_ clear Limpia todo lo que hayamos escrito en la terminal >_ sudo -k o -reset-timestamp invalida el archivo timestamp. >_ sudo -g o -group=group ejecuta comandos como un nombre o ID de grupo especificado. >_ sudo -h o -host=host ejecuta comandos en el host. *Visual Studio Code Ctrl + N: Nuevo archivo Ctrl + O: Abrir archivo Ctrl + S: Guardar archivo Ctrl + Shift + S: Guardar archivo como Ctrl + F: Buscar en archivo Ctrl + H: Buscar y reemplazar en archivo Ctrl + Ñ: Abrir terminal Algoritmo: Es una serie de pasos para resolver un problema.
